Git 高级命令 (动画演示性学习)

· 2020-05-07 ·

一、网站学习

  网站(https://learngitbranching.js.org) 可以说是最好的Git 教程网站，网站内置了Git 沙盒,方便查看每次Git 命令的效果，同时设置了Git 命令从基础到高级的:"闯关游戏"。

二、可视化Git命令

  • git merge(合并)

    区别:
    1、git merge(-ff) 快速推进，如果当前分支与要合并的分支相比，没有额外提交，会快速合并，不创建新的提交。
    2、 git merge(-no-ff) 如果我们在当前分支上提交了要合并的分支没有的更改，git将执行。使用No-fast-foward 合并,No-fast-foward合并，Git在活动分支上创建一个新的合并提交。提交的父提交指向活动分支和要合并的分支！

  • git fetch(拉取)

  • git pull(拉取+合并)

区别：git pull = git fetch + git merge

  • git reset

区别：soft reset 只是将内容回滚到暂存区,工作区内容并未变化，hard reset相当于不保留提交已有的修改，直接将当前分支的状态恢复到某个特定提交下，暂存中或者工作区的内容不复存在。
